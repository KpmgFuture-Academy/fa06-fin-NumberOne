# Tem_fa00-fin-LIVABLE

삼정 Future Academy 최종 프로젝트  
대화형 AI 에이전트 기반 숙소 추천 시스템

---

## 프로젝트 개요

- **프로젝트명**: LIVABLE
- **프로젝트 유형**: 대화형 AI 에이전트 기반 추천 시스템
- **프로젝트 목표**  
  - 사용자의 자연어 대화를 통해 체류 목적·조건·선호를 구조화  
  - LLM 기반 추론과 DB / 외부 검색을 결합하여 즉시 추천 제공  
  - 추천 → 조건 수정 → 재추천 → 행동(RFQ)으로 이어지는 End-to-End 흐름 구현
- **진행 기간**: 2025년 11월 ~ 12월

---

## 요구사항 정의

### 기능 요구사항

- [ ] 사용자 자연어 입력 수집
- [ ] 대화 맥락 기반 사용자 프로필 추출 (JSON 구조화)
- [ ] 숙소 추천 기능  
  - 최소 2개 이상 후보 제시  
  - 추천 근거 명시
- [ ] 조건 수정 요청에 따른 재추천 처리
- [ ] 외부 정보 검색 기능  
  - 지역 특성, 가격 범위, 이동 시간 등
- [ ] 추천 이후 행동 연결  
  - RFQ(견적 요청) 또는 다음 단계 유도

### 비기능 요구사항

- [ ] LLM 오류 및 Tool 실패 시 안전한 Fallback 처리
- [ ] 단일 턴 응답 지연 최소화
- [ ] 도메인 확장이 가능한 구조 설계

## 시스템 아키텍처 개요

- **LLM Agent**
  - 사용자 발화 해석
  - 상태(State) 및 Intent 판단
  - 추천 시점 제어

- **Tool Layer**
  - DB 검색
  - 외부 정보 검색(Web Search)

- **Backend**
  - API 제공
  - 대화 상태 및 로그 관리

- **Database**
  - 숙소 데이터
  - 대화 로그
  - 사용자 프로필

## 모델 정의

### 에이전트 논리 모델

- **State**
  - 탐색 초기
  - 조건 수집
  - 추천 가능
  - 재조정
  - 종료

- **Intent**
  - 추천 요청
  - 조건 제시
  - 조건 수정
  - 거부 / 보류
  - 종료

- **Agent 주요 기능**
  - 사용자 발화로부터 구조화된 정보 추출
  - 추천 필요 시점 판단
  - 적절한 Tool 호출 결정

※ 본 프로젝트에서 “모델”은 ML 학습 모델이 아닌,  
**대화 흐름과 추천 판단을 담당하는 논리적 에이전트 모델**를 의미함
